# MANUAL DO USUÁRIO
## SISTEMA G.T GESTÃO PATRIMONIAL

**Versão:** 2.5 (Enterprise)
**Destinatário:** Secretaria Municipal / Usuários do Sistema

---

## 1. ACESSO AO SISTEMA

### 1.1 Login e Autenticação
O sistema utiliza segurança de nível bancário. Para acessar:
1.  Abra o aplicativo no computador ou celular.
2.  Insira seu **E-mail Institucional**.
3.  Insira sua **Senha** (fornecida pelo Administrador).
4.  Clique em **ENTRAR NO COCKPIT**.

*Nota: Se errar a senha 3 vezes, seu IP pode ser temporariamente bloqueado por segurança.*

### 1.2 Recuperação de Acesso
Caso esqueça a senha:
1.  Na tela de login, clique em **"Esqueci a chave"**.
2.  Digite seu e-mail.
3.  Você receberá um link seguro para redefinição.

---

## 2. GESTÃO DE ATIVOS (PATRIMÔNIO)

### 2.1 Cadastrar Novo Bem
1.  No menu lateral, clique em **Ativos**.
2.  Clique no botão **+ NOVO IMOBILIZADO**.
3.  Preencha a ficha cadastral:
    *   **Nome:** Descrição breve (Ex: Cadeira Giratória).
    *   **Valor:** Valor de aquisição ou reavaliação.
    *   **Localização:** Onde o bem está fisicamente.
4.  Clique em **SALVAR REGISTRO**.
    *   *O sistema grava automaticamente Log de Auditoria desta ação.*

### 2.2 Transferência de Local
1.  Localize o bem na lista.
2.  Clique no ícone de **Editar** (Lápis).
3.  Altere o campo **Localização**.
4.  Confirme a operação.

### 2.3 Baixa Patrimonial
A baixa requer permissão de Supervisor.
1.  No cadastro do bem, selecione a opção **Baixar/Desativar**.
2.  Justifique o motivo (Ex: Obsolescência, Furto, Leilão).
3.  O bem sairá da lista ativa e irá para o "Arquivo Morto" (Auditável).

---

## 3. RELATÓRIOS E AUDITORIA

### 3.1 Emitir Relatório Geral
1.  Vá ao menu **Inteligência** (Reports).
2.  Escolha o tipo: "Inventário Geral" ou "Por Localização".
3.  O sistema gerará um PDF oficial com DATA/HORA e HASH de verificação.

### 3.2 Auditoria (Logs)
1.  Acesse o menu **Auditoria**.
2.  Visualize quem fez o quê, quando e onde.
3.  *Estes registros são imutáveis e servem como prova jurídica.*
